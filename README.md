# منارة الأنبار العاجلة — Anbar Urgent Beacon

نظام الإنذار المبكر لمحافظة الأنبار — تطبيق ويب تقدمي (PWA) للتنبيهات العاجلة والتواصل مع المواطنين.

## المميزات

- تسجيل المستخدمين عبر OTP (WhatsApp)
- تنبيهات عاجلة جغرافية / حسب المدينة / بث عام
- لوحة إدارة متكاملة (إحصائيات، تنبيهات، مستخدمين)
- نظام استفسارات وبلاغات من المواطنين
- مراسلات بين الإدارة والمستخدمين
- تتبع الموقع الجغرافي للمستخدمين
- دعم PWA (تثبيت على الهاتف)
- تصميم عربي RTL مع وضع داكن

## التقنيات

- **Frontend**: React + TypeScript + Vite + Tailwind CSS + shadcn/ui
- **Backend**: Node.js + Express
- **Database**: Supabase (PostgreSQL)
- **Auth**: JWT + OTPiq (WhatsApp OTP)
- **Maps**: Leaflet / react-leaflet

## سجل التحديثات

### 2026-03-23 — إضافة دليل أرقام الطوارئ (Emergency Phone Numbers Directory)

- **صفحة جديدة `/emergency`**: دليل شامل لأرقام هواتف الطوارئ والخدمات في محافظة الأنبار.
- **بانر 911 بارز**: زر اتصال سريع يظهر أعلى الصفحة للدفاع المدني.
- **3 أقسام Accordion قابلة للطي**:
  1. طوارئ وخدمات عامة (الدفاع المدني، بلدية الرمادي، مياه الأنبار)
  2. شكاوى كهرباء مركز الأنبار (الخط الساخن 159، الرمادي، غرب الرمادي، هيت، الخالدية)
  3. شكاوى كهرباء شرق الأنبار (القطاع الشمالي، الجنوبي، الصقلاوية، الكرمة، العامرية)
- **وصول شامل**: الصفحة متاحة قبل التسجيل وبعده (public route).
- **اتصال مباشر**: جميع الأرقام قابلة للنقر عبر بروتوكول `tel:` للاتصال الفوري.
- **تكامل مع التنقل**: أُضيف زر "طوارئ" في شريط التنقل السفلي (BottomNav) والقائمة العلوية (AppHeader).
- **روابط سريعة**: زر "أرقام الطوارئ" ظاهر في الصفحة الرئيسية (Home) وصفحة الهبوط (Index).
- **تصميم عالي التباين**: ألوان طوارئ واضحة (أحمر، برتقالي، أزرق) مع تصميم RTL متجاوب.
