# منارة الأنبار العاجلة — Anbar Urgent Beacon

نظام الإنذار المبكر لمحافظة الأنبار — تطبيق ويب تقدمي (PWA) للتنبيهات العاجلة والتواصل مع المواطنين.

## المميزات

- تسجيل المستخدمين عبر OTP (WhatsApp)
- تنبيهات عاجلة جغرافية / حسب المدينة / بث عام
- لوحة إدارة متكاملة (إحصائيات، تنبيهات، مستخدمين)
- نظام استفسارات وبلاغات من المواطنين
- مراسلات بين الإدارة والمستخدمين
- تتبع الموقع الجغرافي للمستخدمين
- دعم PWA (تثبيت على الهاتف)
- تصميم عربي RTL مع وضع داكن

## التقنيات

- **Frontend**: React + TypeScript + Vite + Tailwind CSS + shadcn/ui
- **Backend**: Node.js + Express
- **Database**: Supabase (PostgreSQL)
- **Auth**: JWT + OTPiq (WhatsApp OTP)
- **Maps**: Leaflet / react-leaflet
